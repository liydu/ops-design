# OPS DSL: Webpack project example

This is an example project uses [OPS DSL](https://github.com/liydu/ops-design)

## Installation

After cloning the repository, install dependencies:

```bash
cd <projectFolder>/ops-design/examples/webpack
npm install
```

Now you can start the local server:

```bash
npm start
```

By default, server is running at `http://localhost:3000`
